# planetearth.live - Modelo de Cuantificación de Impacto

## Fórmula de Impacto Total

Cada actividad se evalúa con esta fórmula:

```
IMPACTO_TOTAL = Impacto_Unitario × Escala × Multiplicador_Red × Palanca_Sistémica × Persistencia
```

Donde:
- **Impacto_Unitario (IU)**: Cambio medible que genera 1 persona haciendo la actividad (1-10)
- **Escala (E)**: Qué tan factible es que 1B personas lo hagan (1-10)
- **Multiplicador_Red (MR)**: Cuánto se amplifica cuando más personas lo hacen (1-5x)
- **Palanca_Sistémica (PS)**: Si cambia reglas upstream que afectan otros sistemas (1-5x)
- **Persistencia (P)**: Cuánto dura el efecto después de la acción (1-5x)

## Análisis de Apalancamiento: Dónde 1 Acción Genera Cascadas

Algunas acciones tienen un efecto dominó desproporcionado:

### Las 5 palancas más poderosas

```
1. EDUCACIÓN SISTÉMICA
   └─> Persona educada
       ├─> Entiende clima -> actúa sobre energía
       ├─> Entiende nutrición -> come mejor
       ├─> Entiende política -> vota informado
       ├─> Entiende finanzas -> invierte responsable
       └─> Enseña a otros -> efecto multiplicador 3-5x
       CASCADA: 1 acción -> 5+ categorías impactadas

2. ELIMINAR SUBSIDIOS A FÓSILES
   └─> Energía renovable se vuelve más barata
       ├─> Transporte eléctrico se acelera
       ├─> Industria limpia se vuelve competitiva
       ├─> Agricultura reduce costos energéticos
       └─> Aire más limpio -> salud mejora
       CASCADA: 1 política -> 5+ sectores transformados

3. REDUCIR CONSUMO DE CARNE 80%
   └─> Menos ganadería industrial
       ├─> -14.5% emisiones GEI
       ├─> 75% menos uso de tierra agrícola
       ├─> 70% menos uso de agua agrícola
       ├─> -100B animales sufriendo/año
       ├─> Menos deforestación -> más biodiversidad
       └─> Mejor salud humana -> menos enfermedades crónicas
       CASCADA: 1 decisión de dieta -> 6+ sistemas mejorados

4. REEMPLAZAR PIB POR BIENESTAR
   └─> Gobiernos optimizan diferente
       ├─> Salud se prioriza sobre producción
       ├─> Naturaleza tiene valor medido
       ├─> Educación se ve como inversión, no gasto
       ├─> Desigualdad se convierte en métrica a reducir
       └─> Éxito = bienestar, no acumulación
       CASCADA: 1 cambio de métrica -> toda la política se reorienta

5. AI ALINEADA CON BIENESTAR PLANETARIO
   └─> AI optimiza para vida, no solo profit
       ├─> Optimiza distribución alimentaria
       ├─> Optimiza redes energéticas
       ├─> Acelera descubrimientos médicos
       ├─> Monitorea biodiversidad en tiempo real
       ├─> Detecta corrupción automáticamente
       └─> Personaliza educación para cada humano
       CASCADA: 1 alineación -> amplifica TODA otra categoría
```

## Principio de Pareto Planetario

Si solo pudieras hacer 3 cosas, estas generan el 80% del impacto:

### Las 3 acciones individuales más poderosas

1. **Cambiar tu dieta** (70%+ plantas, local, sin desperdicio)
   - Impacta: clima, agua, animales, salud, tierra, biodiversidad
   - Se hace 3 veces al día = 1,095 votos/año

2. **Cambiar tu voto económico** (comprar ético, invertir responsable, menos consumo)
   - Impacta: economía, justicia, ambiente, animales, gobernanza
   - Se hace con cada transacción = miles de votos/año

3. **Cambiar tu voto político y educar a otros**
   - Impacta: gobernanza, justicia, todas las políticas públicas
   - Se hace cada día al conversar y cada elección al votar
   - Efecto multiplicador: cada persona que educas amplifica tu impacto
